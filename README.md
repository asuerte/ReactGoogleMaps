<h1>ReactGoogleMaps</h1>

<p>This project was developed with React.js, Express.js, and Node.js that implement the Google Maps API. A user can search for a location in Google maps, then the details of the nearest restaurant or hotel will be displayed.</p>

<h2 id="Installation">Installation</h2>                         
  
1. Clone this repository.
    ```
    git clone https://github.com/asuerte/ReactGoogleMaps.git
    ```
2. Open both GoogleMaps-Frontend and GoogleMaps-Backend in separate source-code editors of your choice.
3. Install project dependencies for both GoogleMaps-Frontend and GoogleMaps-Backend
    ```
    npm install
    ```
4. In the root directory of the GoogleMaps-Backend, start the server.
    ```
    node index.js
    ```
5. In the root directory of the GoogleMaps-Frontend, run the application.
    ```
    npm start
    ```
